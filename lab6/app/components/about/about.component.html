ℹ️ About This Application

Project Overview

This is a full-featured Angular application built as part of a web development course. It combines a Kaspi-style product catalog with a live album browser powered by the JSONPlaceholder REST API.

Features

  • 🛒 Product catalog with categories, image galleries, likes, and sharing
  • 🎵 Album listing with live data from JSONPlaceholder API
  • 🖼️ Per-album photo grid with thumbnail display
  • ✏️ Inline album title editing with save functionality
  • 🗑️ Album deletion with local UI update
  • 🔀 Angular Router with standalone components
